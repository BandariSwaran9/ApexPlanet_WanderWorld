# WanderWorld - Travel Website

Task 1 Submission for ApexPlanet Internship

## Overview

WanderWorld is a responsive multi-page travel website built using HTML5 and CSS3.

## Features

* Responsive Design
* Home Page with Hero Section
* Destinations Page
* Travel Packages Page
* Contact Form
* Google Maps Integration
* Travel Gallery
* Modern UI Design

## Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Media Queries
